# 여섯번째 연습 - 채팅 앱

### UI 구현

[![Chatting](https://github.com/usinuniverse/usinuniverse.github.io/blob/master/assets/images/project%20images/07/01.png?raw=true)](https://vimeo.com/266613688)

* 클릭하시면 동영상을 감상하실 수 있습니다.

### 기술 구현

* 유저의 채팅 메시지 자체가 명령어 처럼 전달(ex. 상담원과 채팅하기 -> 서버를 통해 웹 상담원과 채팅연결, 상담완료 -> 상담원과의 채팅 종료, 그 외 메시지 -> 메시지 형태로 전달)
* 채팅 내역이 생길때마다 채팅 내역을 Realm을 통해 저장.
* 채팅 내역이 생길때마다 마지막 UITableViewCell을 포커싱 하도록 함.
* Socket 통신 구현
* 서버에서 내려오는 메시지는 JSON 형태이며 메시지의 형태나 값에 따라 소켓 연결, 방 입장, 상담원 입장 등 플로우를 설계

### 사용 라이브러리

* Starscream
* SwiftyJSON
* RealmSwift

### 코드

```swift
// 마지막 셀을 바라보도록 하는 메서드
extension UITableView {
    
    func scrollToBottom() {
        let rows = self.numberOfRows(inSection: 0)
        
        if rows > 0 {
            let indexPath = IndexPath(row: rows - 1, section: 0)
            self.scrollToRow(at: indexPath, at: .bottom, animated: true)
        }
    }
}

// 메시지 수신에 따른 처리
func websocketDidReceiveMessage(socket: WebSocketClient, text: String) {
    // 소켓 연결 성공
    let parsedData = JSON(parseJSON: text)
    if parsedData["header"]["action"].stringValue == "connect.success" {
        do {
            // 서버로 보낼 데이터 정의
            let data = try JSONSerialization.data(withJSONObject: ["header":["action":"a100"],"userid":self.username,"name":self.name], options: .prettyPrinted)
            // 데이터 전송
            self.socket.write(data: data)
        } catch {
            print(error)
        }
    }
}

// 메시지 저장 메서드
func saveMessage(text: String) {
    let content = Content(value: ["sender": "나", "message": text])
    
    if realm.objects(ChatModel.self).isEmpty == true {
        // 아직 저장되지 않은 경우
        let chatModel = ChatModel()
        chatModel.contents.append(content)
        
        try! realm.write {
            realm.add(chatModel)
        }
        // 저장되어 있는 경우
    } else {
        try! realm.write {
            // 데이터 업데이트
            let chatModel = realm.objects(ChatModel.self)
            chatModel.first?.contents.append(content)
        }
    }
}
```
