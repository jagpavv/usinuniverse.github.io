# 요새먼지

* [앱스토어 바로가기](https://itunes.apple.com/kr/app/요새먼지/id1336779135?mt=8)

### UI 구현

[![Yosaemunge](https://github.com/usinuniverse/usinuniverse.github.io/blob/master/assets/images/project%20images/06/01.png?raw=true)](https://vimeo.com/252505685)

* 클릭하면 구동 영상을 감상할 수 있습니다.

### 사용법

* 현재 위치 사용시 위치 서비스를 허락하면 자동으로 작동, 수동 위치 사용시 검색 위치를 수동으로 지정
* 관심지역에 현재위치를 바로 추가하거나, 검색하여 추가
* 당겨서 업데이트(pull to refresh)
* 관심지역은 밀어서 삭제, 드래그 앤 드롭을 통해 마음대로 커스텀 가능
* 간소화 보기, 위젯 색상 배경 사용, 현재 위치 사용등 커스텀 가능

### 기술 구현

* 사용자 위치 기반 서비스 구현(사용자 요구로 업데이트)
* 현재 위치 사용 모드, 수동 위치 사용 모드, 간략히 보기 모드 등 사용자화 구현
* 위젯(today extension) 구현
* 서버(에어코리아)의 실시간 자료를 바탕으로 작동(RESTFul api)
* 오픈소스 통해 사용성 개선

### 동작 원리

* 현재 위치 사용시
1. 현재 사용자의 주소를 받는다. (iOS frame work - Core Location)
2. 받아온 주소에서 동, 면, 읍만을 가져온다. (ex. 05113 대한민국 서울특별시 금천구 독상동 1022-6 -> 독상동, 대한민국 충청남도 당진시 고대면 옥현리 315-1 -> 고대면)
3. 가져온 동, 면, 읍을 기준으로 가장 가까운 측정소를 알기 위해 tmX, tmY 좌표를 받아온다. (에어코리아 api)
4. tmX, tmY 좌표를 기준으로 가장 가까운 측정소의 측정값을 받아온다. (에어코리아 api)
5. 측정값을 사용자에게 보여주고, 값에 따른 정보를 나타낸다.

* 수동 위치 사용시
1. 사용자가 현재 위치를 선택한다.
2. 이하 동일
